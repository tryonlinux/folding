# Folding@Home Fan Page (2003)

A website I made as a kid in 2003 to promote the [Folding@home](https://foldingathome.org) distributed computing project and recruit Michigan participants. My dad was super into it at the time and this was my second website ever.

Preserved here as a personal archive.
